# Петли (OBJ-модели)

Сюда кладутся .obj-модели петель.

## Соглашение

- Имя файла = `objFile` из модели фасада. Пример: `wide-profile-hinge.obj`.
- Координаты модели: ось вращения чашки петли = ось Y, центр чашки в начале координат (0,0,0).
- Единицы: миллиметры.

Конфигуратор при загрузке поворачивает/позиционирует модель так, чтобы центр чашки совпадал с центром отверстия присадки.
